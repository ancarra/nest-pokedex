# Ejecutar en desarrollo

1. CLonar el repositorio
2. Ejecutar

```
npm install 
```` 
4. Levantar la base de datos

```
docker compose up -d

```

5. Reconstruir la base de datos de forma masiva

```
https://localhost:4000/api/v2/seed

```

## Stack usado
* MongoDB
* Nest
